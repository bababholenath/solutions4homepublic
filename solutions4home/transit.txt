## If it still loads 5.3.0 — find and remove the culprit

The old ADFS packages that typically drag in 5.3.0 are:
```
Microsoft.Owin.Security.WsFederation   ← likely culprit
System.IdentityModel                    ← old WIF
Microsoft.IdentityModel (old 1.x)       ← old WIF
```

Remove them if you no longer need ADFS:
```
PM> Uninstall-Package Microsoft.Owin.Security.WsFederation -RemoveDependencies
PM> Uninstall-Package Microsoft.IdentityModel -RemoveDependencies
```
